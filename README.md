## 📊 Data Analytics Internship Projects - Dynamix Networks

## 📌 Project 1: Netflix Movies & TV Shows Analysis

### Objective:
To analyze Netflix content and extract insights about trends, genres, and distribution.

### Key Tasks:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Advanced Analysis

### Key Insights:
- Netflix has more Movies than TV Shows
- Content growth increased after 2015
- USA produces most content
- Drama and International genres dominate
- Content addition peaked after 2016.
